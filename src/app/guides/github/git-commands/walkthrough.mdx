2.  **Initialize local repo**

```bash
git init
```

3.  **Stage README for commit**

```bash
git add README.md
```

4.  **First commit**

```bash
git commit -m "Initial commit"
```

5.  **Rename default branch to 'main'**

```bash
git branch -M main
```

6.  **Add remote GitHub repo**

```bash
git remote add origin https://github.com/JxTorres003/Project_React.git
```

7.  **Push to GitHub and track**

```bash
git push -u origin main
```
